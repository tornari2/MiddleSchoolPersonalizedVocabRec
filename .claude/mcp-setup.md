# MCP Server Configuration

This project uses the following MCP servers:

## taskmaster-ai

A task management MCP server that helps with organizing and tracking tasks.

**Configuration:**
```json
{
  "command": "npx",
  "args": ["-y", "task-master-ai"]
}
```

**Location:** `~/.config/claude/mcp.json`
